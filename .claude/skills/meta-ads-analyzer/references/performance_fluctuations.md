# Performance Fluctuations

## Overview

Performance fluctuations are normal in Meta Ads. Understanding which fluctuations are expected vs concerning helps avoid unnecessary changes that can hurt performance.

## Normal Fluctuations

### Day-to-Day Variation
- **CPA can vary 20-50% day-to-day** — this is normal
- Evaluate performance over **7-day rolling averages**, not daily snapshots
- Weekday vs weekend patterns are common and expected

### Weekly Cycles
- Many industries see predictable weekly patterns
- B2B: Higher performance Mon-Fri
- B2C/E-commerce: Spikes on weekends or paydays
- These patterns reflect user behavior, not system issues

### Seasonal Effects
- Holidays, events, and seasons affect competition and user behavior
- Q4 (Oct-Dec) typically has higher CPMs due to holiday advertising
- Plan for these changes rather than reacting to them

### After Edits
- Any significant edit triggers a recalibration period
- Expect 24-48 hours of unstable performance after changes
- Budget changes >20% may reset the learning phase entirely

## Concerning Fluctuations

### Sustained CPA Increase (7+ Days)
- **When to worry:** CPA trending upward for more than 7 consecutive days
- **Possible causes:**
  - Audience fatigue (high frequency)
  - Increased competition
  - Creative fatigue
  - Seasonal cost increases
  - Landing page issues

### Sudden Delivery Drop
- **When to worry:** Spend drops to near zero unexpectedly
- **Possible causes:**
  - Budget depleted
  - Account or ad disapproved
  - Audience too narrow after targeting changes
  - Bid cap too restrictive
  - Payment method issues

### Frequency Creep
- **When to worry:** Frequency exceeding 3-4x for cold audiences
- **Impact:** Ad fatigue leads to declining engagement and rising costs
- **Solution:** Refresh creative, expand audience, or rotate ads

## Diagnostic Framework

### Step 1: Check the Time Window
- Is this a 1-day blip or a sustained trend?
- Compare to 7-day and 28-day averages
- Look at year-over-year data if available

### Step 2: Check for External Factors
- Was there a significant edit recently?
- Is there a holiday or seasonal event?
- Has competition changed (check CPM trends)?

### Step 3: Check Internal Metrics
- **Frequency:** Is it too high? (audience fatigue)
- **CTR trends:** Declining CTR suggests creative fatigue
- **Relevance diagnostics:** Check quality, engagement, conversion rankings
- **Conversion rate:** Declining conversion rate may indicate landing page or audience issues

### Step 4: Determine Action

| Situation | Action |
|-----------|--------|
| 1-2 day fluctuation, within normal range | No action — monitor |
| Post-edit volatility (< 3 days) | Wait for stabilization |
| High frequency (>3x cold audience) | Refresh creative, expand audience |
| Sustained CPA increase (7+ days) | Diagnose root cause, test changes |
| Creative fatigue (declining CTR) | Introduce new creative variations |
| Sudden delivery drop | Check account status, budget, audience |

## Best Practices

1. **Don't react to daily noise** — Use 7-day windows for decisions
2. **Set up alerts for sustained trends** — 7+ days of directional change
3. **Keep a change log** — Track all edits to correlate with performance shifts
4. **Have creative ready** — Plan creative refreshes before fatigue sets in
5. **Use automated rules carefully** — Rules based on single-day performance can cause over-optimization
